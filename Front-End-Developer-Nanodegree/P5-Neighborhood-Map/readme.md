## Requires Python to be installed
### Mac and Linux: Terminal
Run command inside P5 NEIGHBOURHOOD MAP directory
    python -m SimpleHTTPServer 8000

### Windows CMD
Run command inside P5 NEIGHBOURHOOD MAP directory
    python -m http.server 8000

### Navigate in browser
http://localhost:8000/
